# Mike Anthony - Professional CV Site

This is the source code for the professional portfolio and curriculum vitae of Mike Anthony, Audit Professional specializing in IT & Government Performance.

## Tech Stack
* Vanilla HTML5
* CSS3 (Custom Properties, Flexbox, Grid)
* Vanilla JavaScript (Category filtering and smooth navigation)
* Zero-build deployment via Netlify

## Features
* **Responsive Layout**: Designed for seamless viewing across desktop and mobile.
* **Interactive Timeline**: Professional history presented in an expandable, readable format.
* **Skill Filtering**: Viewers can instantly filter experience based on specific domains (Audit, Operations, Projects).

## Deployment
This static site is configured to automatically deploy to [mikeeanthony.com](https://mikeeanthony.com/) via Netlify whenever updates are pushed to the `main` branch.
